# React + TypeScript + Vite

### Clone the project

```bash
git clone https://github.com/fastigor293015/f2p-games.git
```

### Enter the project directory

```bash
cd f2p-games
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm start
```

### Create bundle for production

```bash
npm run build
```
